Test newsletter

To test the internal newsletter use the urls
  • http://{website}/TestNewsletter/int
  • http://{website}/TestNewsletter/int/{year}/{month}/{day}/
Examples:
  • /TestNewsletter/int
  • /TestNewsletter/int/2011/12/02/
To test the external newsletter use the urls
  • http://{website}/TestNewsletter/ext
  • http://{website}/TestNewsletter/ext/{year}/{month}/{day}/
Examples:
  • /TestNewsletter/ext
  • /TestNewsletter/ext/2011/12/02/

Test tomorrow events

To test tomorrow events newsletter, use the urls
  • http://{website}/TestEvent/
  • http://{website}/TestEvent/{year}/{month}/{day}/
Examples:
  • /TestEvent/
  • /TestEvent/2011/12/02/
