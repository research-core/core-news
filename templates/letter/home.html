Test newsletter

To test the internal newsletter use the urls
  • http://{website}/test_newsletter/int
  • http://{website}/test_newsletter/int/{year}/{month}/{day}/
Examples:
  • /test_newsletter/int
  • /test_newsletter/int/2011/12/02/
To test the external newsletter use the urls
  • http://{website}/test_newsletter/ext
  • http://{website}/test_newsletter/ext/{year}/{month}/{day}/
Examples:
  • /test_newsletter/ext
  • /test_newsletter/ext/2011/12/02/

Test tomorrow events

To test tomorrow events newsletter, use the urls
  • http://{website}/test_event/
  • http://{website}/test_event/{year}/{month}/{day}/
Examples:
  • /test_event/
  • /test_event/2011/12/02/
